================================================================================
          AI PROMPTS FOR GENERATING EASY-TO-SLICE SPRITE SHEETS (SINGLE-IMAGE)
================================================================================

This file contains copy-pasteable prompt templates for ChatGPT (DALL-E) or other 
AI image generators. These are optimized so the "Auto-Detect Grid & Borders" 
feature in Image Grid Slicer can parse the sheet easily.

--------------------------------------------------------------------------------
PROMPT 1: STANDARD SPRITE SHEET (Transparent or Solid Background)
--------------------------------------------------------------------------------
Use this prompt when you want a standard sprite sheet without complex backgrounds:

"Generate a 2D game sprite sheet of [DESCRIBE CHARACTER/ITEM, e.g., a pixel art 
wizard walking]. 
Format the image as a uniform grid of 3 rows and 4 columns (12 frames total).
INSTRUCTIONS FOR AUTO-DETECTION:
1. The background of the entire image must be a single, solid flat color (such 
   as pure white or pure black) with no gradients, lighting, or shadows.
2. Leave a clear, uniform spacing gap (gutter/margin) of about 15-20 pixels 
   between all rows and columns.
3. Every sprite must be identical in size, perfectly centered in its grid cell, 
   and must not touch the borders or other sprites."

--------------------------------------------------------------------------------
PROMPT 2: SPRITE SHEET WITH COMPOSITED BACKGROUND (Single Image Output)
--------------------------------------------------------------------------------
Use this prompt when you want your animation frames to overlay on a background. 
Instead of generating the background separately, this prompt tells ChatGPT to 
output the background as the first frame (Frame 1) of the grid, followed by the 
sprites drawn on top of it.

"Generate a single 2D game sprite sheet formatted as a uniform grid of 3 rows and 
4 columns (12 frames total). 
The theme/scene is: [DESCRIBE THE SCENE/CHARACTER, e.g., a futuristic robot 
running inside a spaceship corridor].
INSTRUCTIONS FOR THE CELL GRID:
- Cell 1 (Row 1, Column 1) must be the background scene ONLY (empty spaceship 
  corridor, no character, no sprite).
- Cells 2 through 12 (the remaining 11 frames) must contain the character/sprite 
  performing its animation sequence, layered directly on top of the exact same 
  background corridor.
- Each cell must have identical dimensions and the background must align 
  perfectly across all cells.
- Leave a clear, uniform solid-color spacing gap of about 15-20 pixels between 
  all cells so the auto-slicer can detect the grid layout."

--------------------------------------------------------------------------------
💡 HOW TO ANIMATE USING THE COMPOSITED BACKGROUND FRAME:
--------------------------------------------------------------------------------
1. Load the generated grid image into Image Grid Slicer.
2. Click "Auto-Detect Grid & Borders" and click "Slice Image".
3. Go to the "Animate" tab and click "Load Frames from Slice".
4. Check the box "Use background image".
5. Set "Background Frame Index" to 1 (this corresponds to Cell 1, which contains 
   only the background scene).
6. The preview will automatically composite the sprites from the other frames 
   centered over the background frame, creating a smooth animation!
